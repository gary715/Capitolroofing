## Tapered Insulation Details (Section 07 22 00)
- **Slope:** 1/8" per foot
- **Min thickness at drain (low point):** 1.5"
- **R-value target:** R-30 average
- **Panel size:** 4x4 (this job — some jobs use 4x8)
- **Attachment:** Solvent-free insulation adhesive at 1.5 gal per 100 SF
- **Stagger joints:** 6" minimum in both directions between layers
- **Cover board:** 1/2" HD wood fiberboard adhered over tapered ISO

### Tapered Panel Thickness Sequence (1/8" per foot)
Each foot from drain = 1/8" rise. Panels run 4 feet = 1/2" rise per panel.

| Distance from drain | Cumulative rise | Panel min thickness |
|---|---|---|
| 0–4 ft | 0–0.5" | 1.5" (minimum) |
| 4–8 ft | 0.5–1.0" | 2.0" |
| 8–12 ft | 1.0–1.5" | 2.5" |
| 12–16 ft | 1.5–2.0" | 3.0" |
| 16–20 ft | 2.0–2.5" | 3.5" |
| 20–24 ft | 2.5–3.0" | 4.0" |
| 24–28 ft | 3.0–3.5" | 4.5" |
| etc. | | |

> Note: Actual panel thicknesses depend on tapered plan layout — upload the tapered plan for exact panel counts.
