# Estimating Rules — Capitol Roofing

## System Type
- We install **IB Roof Systems PVC** flat roofing only
- Two job types:
  1. **Tear-off & Replace** — remove existing roof down to deck, install new system
  2. **Overlay (Recover)** — install new IB membrane directly over existing roof

## Measurement Rules

- Measure the **actual roof area** in square feet, then divide by 100 to get squares
- Always round **up** to the nearest half square
- Include all sections — main field, equipment pads, raised sections
- Do NOT deduct for drains or small penetrations under 4 sq ft

## Logistics Factors

These factors affect labor cost and must be assessed on every job:

### Building Height
| Stories | Access Factor |
|---------|--------------|
| 1 story | Standard — no adjustment |
| 2 stories | Add **[BOSS TO FILL]** per square |
| 3+ stories / ladder access only | Add **[BOSS TO FILL]** per square |

**Rule:** If no roof hatch and no internal access, count as ladder-access job.

### Material Walk Distance
- Measure the approximate walking distance from where the truck parks to where materials are loaded onto the roof
- **Under 50 ft** — no adjustment
- **50–150 ft** — add **[BOSS TO FILL]** per square
- **Over 150 ft** — add **[BOSS TO FILL]** per square + flag for crew discussion

### Foot Traffic / Urban Job Site
- If the job is in a busy area (pedestrian traffic, street access only, no staging area):
  - Add **[BOSS TO FILL]** to the total for site management
  - Flag if sidewalk protection or pedestrian barriers are required
  - Flag if a city permit may be needed for street use
