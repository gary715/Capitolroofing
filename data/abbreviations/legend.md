## IB PVC Clad Metal (Sheet Metal Types)

All metal listed below is **IB PVC clad metal** unless a gauge (e.g. .032) is specified for non-PVC-clad aluminum.
PVC clad metal heat-welds directly to the IB field membrane.

| Abbreviation | Meaning | Notes |
|---|---|---|
| EDG / DE | Drip Edge (edge metal) | Perimeter edge termination |
| GS / GSx | Gravel Stop — number = face height in inches (e.g. GS5 = 5" gravel stop) | Raised edge metal |
| COP / Cap | Coping cap | Covers top of parapet wall — profile includes face + top + back leg |
| REG | Reglet | Counterflashing at walls |
| STEP | Step flashing | At vertical wall intersections |
| VLY | Valley metal | At roof valleys / transitions |
| BASE | Base flashing | At walls and curbs |
| Term bar | Termination bar | Mechanically fastened at edges and walls |
| Pitch Pocket | Pitch pocket — metal box | Heights noted (e.g. "5" High") |

### Metal Gauge Notation
- **.032** = .032 gauge aluminum (thicker, structural applications — coping caps, large drip edges)
- **.024** = .024 gauge aluminum (standard weight)
- When gauge is listed (e.g. ".032 DE 4x4 kick"), the number before the type is the aluminum thickness

### Metal Profile Notation
- **4x4** = 4 inch face x 4 inch flange
- **5 1/2"** = 5.5 inch width (e.g. coping cap face)
- **"plus kick"** = a bent lip at the bottom edge of drip edge to direct water away from wall
- **"High"** = vertical height of metal (e.g. pitch pocket "5" High")

**Important:** When any IB PVC clad metal is listed, the cover strip rule applies — see Derived Materials Rules.

## Rules for Using This Legend

1. Every abbreviation on a field sheet **must** resolve to an item in this legend
2. If an abbreviation is not found, flag it for review — do NOT assume
3. If a symbol appears without a number, return to the field sheet and look for a nearby note
4. Pipe counts must be exact — confirm total before generating material list
5. Any item marked with ★ must appear in the estimate notes section
6. Metal listed without LF must be flagged — **LF is required to calculate cover strip**
7. "Re-Roof" alone does not confirm job type — check work order or notes for overlay vs tear-off
8. Multiple roof sections in a sketch are additive — sum all sections for total squares
9. If total squares in sketch does not match total squares in work order, flag the discrepancy
